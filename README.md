# RMC - Remove Class by Cookie

## Install NPM packages

```
$ npm install
```

## Build and Dist

```
$ grunt
```

## Using

Copy and paste:

```html
<script type="text/javascript" src="//yourserver/path/rmc.min.js"></script>
<script type="text/javascript">
(function() { rmc('cookie-name', 'class-to-remove'); })();
</script>
```
